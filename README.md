## Cách chạy dự án

Yêu cầu:

- Đã cài đặt Flutter SDK
- VS Code hoặc Android Studio
- Máy ảo hoặc thiết bị thật

Các bước thực hiện:

Clone project:

git clone https://github.com/your-username/flutter_calculator_yourname.git

Di chuyển vào thư mục project:

cd flutter_calculator_yourname

Cài đặt thư viện:

flutter pub get

Chạy ứng dụng:

flutter run

## Chức năng

Chức năng cơ bản:

- Cộng (+)
- Trừ (-)
- Nhân (×)
- Chia (÷)
- Xóa toàn bộ (C)
- Nhập số thập phân (.)
- Đổi dấu (+/-)
- Tính phần trăm (%)
- Xử lý lỗi chia cho 0

Chức năng giao diện:

- Giao diện responsive sử dụng Expanded và flex
- Hiển thị kết quả căn phải
- Màn hình hiển thị 2 dòng:
  - Dòng trên: phép tính
  - Dòng dưới: kết quả

- Bố cục nút theo thiết kế Figma
- Khoảng cách giữa các thành phần chuẩn 16px
- Nút bấm được thiết kế đẹp, dễ nhìn
